# Changelog

## 2026-05-25

- Добавлен статический интерфейс Reqcore для подбора персонала: вакансии, кандидаты, резюме, оценка соответствия, критерии оценки, ранжирование, настройки, профиль и администрирование.
- Добавлена русская локаль `ru-RU` и подключён русский язык как язык интерфейса по умолчанию.
- Добавлены npm-скрипты для линта локализации, сборки, smoke-test и браузерной проверки со скриншотами через Chrome.
- Документирован локальный запуск интерфейса и путь к артефактам проверки.
- Сохранены скриншоты ключевых страниц подбора и DOM-отчёт проверки отсутствия непереведённых пользовательских строк.
- Убрано непонятное англоязычное сокращение из пользовательского интерфейса и добавленной документации; вместо него используются понятные русские формулировки по контексту.
