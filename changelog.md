# Changelog

## 2026-05-25

- Создан основной консолидированный документ ВКР `documentation/master-thesis-obsidian/thesis-draft/ВКР.md`; в него перенесены академически корректные формулировки по сравнению результатов модели, включая majority-class baseline `accuracy = 0.56`, результат BERT `accuracy = 0.75`, `MCC = 0.550` и ограничения интерпретации.
- В разделе `2.1` ПЗ уточнено описание преимуществ BERT: формулировка заменена на осторожное утверждение о предпосылках для более точного сопоставления и необходимости экспериментальной проверки относительно обоснованных контрольных подходов.
- В разделе `2.2` ПЗ добавлено объективное сравнение результатов модели с majority-class baseline, рассчитанным по фактическому распределению классов итогового датасета (`accuracy = 0.56`), вместо использования произвольного ориентира.
- В разделе `2.2` ПЗ уточнена интерпретация `accuracy = 0.75` и `MCC = 0.550`: добавлены ограничения вывода и указана необходимость сравнения с дополнительными baseline-методами.
- В разделе `2.3` ПЗ смягчены выводы о влиянии автоматизации на процесс рекрутмента: категоричные утверждения заменены на академически корректные формулировки о потенциальном эффекте и необходимости дополнительной проверки.
- Отдельные файлы презентации в репозитории не обнаружены; правки внесены в доступные Markdown-разделы пояснительной записки.
