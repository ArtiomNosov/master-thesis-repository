# Часто используемые команды

## Git команды

### Быстрый коммит (однострочник)
Используйте эту команду для быстрых коммитов, когда обычный `git commit` не работает из-за блокировки файла:

```powershell
$msg = "Add detailed NIR assignment documentation in Russian for the candidate ranking module, outlining tasks, deadlines, and literature references."; $branch = git branch --show-current; $tree = git write-tree; $parent = git rev-parse HEAD; $commit = git commit-tree -m $msg -p $parent $tree; $commit | Out-File -FilePath ".git\refs\heads\$branch" -Encoding ASCII -NoNewline -Force; git log --oneline -1
```
