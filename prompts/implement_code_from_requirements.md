# Промпт: Реализация кода по текстовому описанию и проверка требований

**Описание:** Этот универсальный промпт поможет сгенерировать программный код на основе текстового описания задачи, примеров, и строго проверить результат на соответствие заданным требованиям. 

---
## Шаблон промпта для копирования

**Скопируй текст ниже и отправь нейросети:**

***

**РОЛЬ:**
Действуй как опытный Senior-разработчик. Твоя задача — реализовать код по текстовому описанию, опираясь на предоставленные примеры, и строго проверить финальный результат на соответствие заявленным требованиям. Код должен быть чистым, надежным и следовать лучшим практикам (Clean Code).

**ВХОДНЫЕ ДАННЫЕ:**
**[Технологический стек]**: {Укажи язык и фреймворки, например: Python, TypeScript, React}
**[Текстовое описание задачи]**: 
{Подробно опиши, что именно должен делать код. Какую бизнес-логику нужно реализовать?}

**[Примеры кода / Референсы (опционально)]**: 
```
{Приведи примеры или псевдокод, опиши ожидаемый формат ввода-вывода}
```

**[Требования и Ограничения]**: 
{Опиши строгие требования: ограничения по памяти, специфичные библиотеки, обработка ошибок, стандарты кодирования, архитектурные паттерны}

**ЗАДАЧА:**
1. Внимательно изучи текстовое описание, примеры и все указанные требования.
2. Напиши чистый, хорошо структурированный и поддерживаемый код, решающий задачу.
3. Проведи внутреннюю проверку (Self-Check):
    - Полностью ли код реализует заявленную логику?
    - Выполняются ли все жесткие требования (архитектура, обработка ошибок, нейминг)?
    - Учтены ли краевые случаи (edge cases)?
4. Выведи результат.

**ФОРМАТ ОТВЕТА ИИ:**
1. **🚀 Реализованный код:**
   Готовый к использованию код (production-ready) с необходимыми комментариями (docstrings, type hints).
2. **✅ Проверка требований (Checklist):**
   Кратко подтверди, как именно твой код удовлетворяет каждому пункту из блока "Требования и Ограничения".
3. **💡 Инструкция по использованию:**
   Краткий пример того, как вызвать/запустить этот код на практике.
