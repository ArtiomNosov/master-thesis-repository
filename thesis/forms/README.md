# Official Forms

Put official department PDFs here before final submission:

- `title.pdf` - official title page.
- `title-dep22.pdf` - official ВКР signature sheet.
- `task.pdf` - official assignment sheet.

The LaTeX entrypoint `thesis/latex/master-thesis-3-pz.tex` checks these paths. If a file is missing, it uses a visible TODO placeholder so the draft remains buildable but cannot be mistaken for the final submission copy.
