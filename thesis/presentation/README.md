# Presentation Template

The file `uir-nir-vkr-wide-template-v2.pptx` is the provided 16:9 department template/methodology deck.

Extracted structure:

1. Service slide with methodology requirements; do not include in the final defense deck.
2. Title slide.
3. Abstract.
4. Relevance.
5. Goal.
6. Analysis results.
7. ВКР tasks.
8. Modeling results.
9. Design results.
10. Implementation results.
11. Conclusion.
12. References.
13. Approbation.
14. Thank-you slide.

Submission constraints from the template:

- Final presentation: 15-20 slides.
- Talk: 7-10 minutes.
- Minimum text, maximum tables/graphics/diagrams.
- Mostly textual slides are acceptable only for title, abstract, goal, tasks, conclusion, references, and approbation.
- Print handouts: 5 copies.
